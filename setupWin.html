Installation Procedure

  1. Download apache and extract it
  2. Move the apache24 folder to your C: drive.
  3. Download Microsoft visual C++. Download only the file that has the same bit for the laptop.
    • 64 bit = x64
    • 32 bit = x86
  4. After downloading Microsoft Visual C++ , double click on it to do the installation.
  5. Launch CMD as administrator
  6. Type “C:\apache24\bin\httpd -k install” . ( If there is error mentioning you do not have any specific dll file , download microsoft visual C++ 2012/2015 should be able to solve it. )
  7. Right click on the Window Button and open System.
  8. Go “Advanced System Setting” > “Advance” Tab > “Environment Valuables”
  9. Go to “Path” Under “System Variable”.
  10. Add these new variable /file path
    • C:\php
    • C:\Apache24\
    • C:\Apache24\bin
  11. Restart laptop and make sure apache starts automatically.
