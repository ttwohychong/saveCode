Setup

Install Vue CLI
  1. Download Node.js
  2. Open cmd
  3. npm install -g @vue/cli
Checkout vue base project
  1. Open cmd
  2. cd to member folder
  3. npm install
  4. npm run serve

Page Template

Sample page template for vue

<template> <div> <HelloWorld msg="Welcome to Your Vue.js App"/>
<div>{{message}}</div>
<div @click="sampleFunction">Use @click for click function, @ is shorthand of v-on</div>
<div v-if="showBlock">if showBlock == true then show this block</div>
<div :data-name="name">In this case data-name = "abc" (based on variable declared in data), : is shorthand for v-bind</div>
<div data-name="name">If data-name didn't bind with any variable using :, it's a String, so in this case data-name="name"</div>
<input v-model="textInp" />
</div> </template>

<script> // @ is an alias to /src
import HelloWorld from '@/components/HelloWorld.vue'
export default { name: 'Home',
data () { return { message: "Test", message2: "Test 2", showBlock: false, name: "abc", textInp: "abc", } },
computed: {},
components: { HelloWorld },
created: function(){ // window onload }, mounted: function(){ // document ready }, methods: { sampleFunction: function() { // your code } } } </script>

If Else Condition (v-if)

Script

export default { name: 'customPagination', data () { return { isShow: false, num: 10 } }, }

Template

<div> <div v-if="isShow">Show this block if isShow == true</div> <div v-else>else show this block</div> </div>
<div> <div v-if="num < 0">Show this block if num < 0</div> <div v-else-if="num > 10">Show this block if num > 10</div> <div v-else>if not fullfilled other conditional statement, default show this</div> </div>

For Loop (v-for)

Script

export default { name: 'customPagination', data () { return { array: [ "Item 1", "Item 2", "Item 3", "Item 4", "Item 5", "Item 6", "Item 7", ], } }, }

Template

Like jQuery / vanilla js foreach function

<div v-for="(v, k) in array" :key="k">Key: {{k}}, Value: {{v}}<div>
