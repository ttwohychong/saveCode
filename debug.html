Debug

To debug, you may:

  1. Check console log see whether got error message
  2. Check superadmin see api is it call successfully (and make sure data pass correctly)
  3. Check reqDefault have the command or not
  4. Check data you pass is it same with trello
  5. Grab latest language file from server using scp

Uncaught SyntaxError: Unexpected identifier

  • Check every object comma / semicolon put correctly

When click button, page no effect (no hang)

  • Make sure all error field display correctly based on backend return

Filter not working

  • Make sure pass searchData instead of inputData
  • dataName = username, dataType = match / like instead of text
  • Go superadmin to make sure searchData pass correctly

When click button page hang, keep show loading icon

  • Make sure language js is up to date
  • Make sure error field display correctly

After apply numberThousand function, showing NaN.00

  • Try parseFloat first for number then only apply numberThousand
  • Make sure backend the data you get is not -, is yes, should do a condition if data == '-', number = 0

Uncaught TypeError: Cannot read property 'variable' of undefined

  • Make sure the variable is declare correctly (var variable;)
  • If the variable declared global, make sure when assign variable data in function didn't declare again the variable. Use variable = 1 instead of var variable = 1 in function
var declaredVar;$(document).ready(function(){});function usedFunc() { declaredVar = YOUR_DATA; var declaredVar = YOUR_DATA; }
